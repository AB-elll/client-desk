# ClientDesk

クライアント企業の経営情報を常時完全管理するデータルームシステム

> Always DD-ready. Every client, every category.

## 概要

ClientDeskは、契約・許認可・従業員・資産など企業経営に関わる全情報を構造化して管理し、M&Aデューデリジェンスにいつでも対応できる状態を維持するOSSシステムです。

```
DriveDesk（書類自動処理）
    ↓
ClientDesk（台帳管理・期限アラート・DDエクスポート）
    ↓
Googleスプレッドシート / Telegram通知
```
